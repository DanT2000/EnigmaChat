## üîê EnigmaChat Logic

- üì• **If a chat with that name already exists**, you'll be asked to enter the **access password**.
- üÜï **If the chat doesn't exist**, it will be created automatically. The **first message** you send will become the **password for that chat** (you‚Äôll need it next time to access it).
